HTML Cheat Sheet

Basic Tags

<html></html>
The HTML tag creates an HTML document.
<head></head>
The head tag holds information about the website as well as links to external resources.
<body></body>
The body tag holds the visible portion of the document. This is where your content will live.
<div></div>
The div tag creates a block container element.
<span></span>
The span tag creates an inline container element.

Images

<img />
The image tag, or img tag, is used to display an image. Requires the src attribute, which identifies the source of the image and the alt attribute, which provides alternative text. This is a self-closing tag.
<img src="image.jpg" alt="Image Description" />

Links

<a></a>
The a tag, or anchor tag, creates a link. Requires the href attribute, which tells the link where to go. May include the title attribute, which provides additional information about the link.
<a href="http://website.com" title="Link Description">Clickable Text</a>

Text Formatting

<h1></h1>
<h2></h2>
<h3></h3>
<h4></h4>
<h5></h5>
<h6></h6>
Heading tags create a hierarchy of headings with h1 being the largest and h6 being the smallest.
<h1>Heading Text</h1>
<em></em>
Adds emphasis to a word (italic).
<em>Italicized Text</em>

Text Formatting, Continued

<strong></strong>
Adds emphasis to a word (bold).
<strong>Bold Text</strong>
<p></p>
A paragraph tag or p tag creates a paragraph element.
<p>A paragraph of text. Chupa chups topping cake tiramisu liquorice sweet roll brownie. Jelly souffle bear claw. Carrot cake jujubes chupa chups oat cake tart.</p>
<blockquote></blockquote>
For quotes, indents text from both sides.
<blockquote>A text quote. Chupa chups topping cake tiramisu liquorice sweet roll brownie. Jelly souffle bear claw. Carrot cake jujubes chupa chups oat cake tart.</blockquote>
<li></li>
Creates a list item. Used in either an ordered lists or an unordered lists.
<li>List Item</li>

Text Formatting, Continued

<ol></ol>
An ordered list, creates a numbered list.
<ol>
  <li>List Item</li>
  <li>List Item</li>
</ol>
<ul></ul>
An unordered list, creates a bulleted list.
<ul>
  <li>List Item</li>
  <li>List Item</li>
</ul>
<br />
A break tag inserts a line break in a chunk of text. Often used inside of a paragraph tag. This is a self-closing tag.
<p>A paragraph of text<br />
split into two lines.</p>
<hr />
A horizontal tag inserts a horizontal rule (line). This is a self-closing tag.
<p>A paragraph of text
<hr />
split into two lines with a line between them.</p>
